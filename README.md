# Warning Overlay Component

A customizable warning overlay component featuring an animated caution tape border, blur effect, and smooth transitions. Perfect for displaying maintenance notices, version updates, or important announcements on your website.

## Features

- Responsive design that works seamlessly across all device sizes
- Smooth fade-in and fade-out transitions
- Warning tape border effect with diagonal stripes
- Backdrop blur effect
- Mobile-friendly layout
- Button hover animations

## Installation

1. Copy the CSS styles (everything between `<style>` tags) from the source code
2. Copy the overlay HTML structure (the `div` with class `warning-overlay` and its contents)
3. Customize the content inside `warning-content` to match your needs

## Customization

### Colors
- Warning tape stripes: Modify the gradient colors
- Content background
- Text color
- Button colors and hover states

### Dimensions
- Content width
- Padding
- Button size
- Tape stripe width

### Animation
The fade animation speed can be adjusted in the `.warning-overlay` transition property.

## Browser Support

Works in all modern browsers that support:
- CSS Grid
- Flexbox
- CSS Transitions
- Backdrop Filter

## Usage Tips

1. Place the overlay HTML just before your closing `</body>` tag
2. Use unique IDs if you need multiple overlays
3. For older browsers, the background opacity serves as a fallback for the blur effect
